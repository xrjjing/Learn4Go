# 快速开始（Go 版）

## 运行示例
- 基础入口：`go run ./cmd/learn4go`
- TODO API：`go run ./cmd/todoapi`（默认 8080）
- 批量重命名：`go run ./cmd/batchrename --dir=. --suffix=.txt --prefix=new_ --apply`
- 日志归档 ZIP：`go run ./cmd/logzip`

## 质量检查
- `GOCACHE=$(pwd)/.gocache go test ./...`
- `make fmt vet test`（如未安装 golangci-lint，lint 会提示跳过）

## 前端联动
- 静态资源在 `web/`
- `web/config.js` 中 `todoApiBaseUrl` 指向 8080 的 Go TODO 服务；可通过 URL `?mock=true` 启用前端 mock

## 依赖与网络
- 当前示例全部基于标准库，无需外网拉取依赖
- 未来引入 Gin/Fiber/gRPC 时再执行 `go mod tidy`
