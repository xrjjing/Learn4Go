# 06 结构体、方法与组合

## 关键点
- 方法接收者：值接收者 vs 指针接收者
- 组合：匿名字段（嵌入）代替继承
- 结构体标签：序列化/校验

## 示例
```go
type User struct {
    ID   int    `json:"id"`
    Name string `json:"name"`
}

func (u *User) Rename(name string) {
    u.Name = name
}
```

### 组合
```go
type Logger struct{ Prefix string }
func (l Logger) Log(msg string) { fmt.Println(l.Prefix, msg) }

type Service struct {
    Logger
}

s := Service{Logger: Logger{Prefix: "[svc]"}}
s.Log("start")
```
