# 09 包与模块管理

## 模块
- 初始化：`go mod init <module>`
- 整理依赖：`go mod tidy`
- GOPROXY 可设为 `https://goproxy.cn,direct`（视网络策略）

## 包
- 目录即包；文件需相同 `package` 声明
- 导出规则：标识符首字母大写

## 示例
```
module github.com/you/demo
require (
    # 依赖列表由 go mod tidy 生成
)
```

## 练习
- 创建子包 `mathx`，实现 `Max/Min`，在 main 中调用
- 运行 `go list ./...` 查看模块内包列表
