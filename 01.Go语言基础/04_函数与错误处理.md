# 04 函数、多返回值与错误处理

## 关键点
- 支持多返回值与命名返回值
- 变参：`func sum(nums ...int) int`
- 错误为值：`error`
- 延迟执行：`defer`，LIFO
- panic/recover：仅在不可恢复场景使用

## 示例
```go
func div(a, b float64) (float64, error) {
    if b == 0 {
        return 0, fmt.Errorf("divide by zero")
    }
    return a / b, nil
}

func main() {
    defer fmt.Println("done")
    if r, err := div(6, 2); err != nil {
        log.Fatal(err)
    } else {
        fmt.Println(r)
    }
}
```
