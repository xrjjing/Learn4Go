# 03 流程控制

## if/else
- 条件无需括号，支持短变量声明：`if v := f(); v > 0 {}`

## for
- 唯一循环关键字：`for init; cond; post {}`
- while 风格：`for cond {}`；无限循环：`for {}`

## switch
- 默认不贯穿，使用 `fallthrough` 显式继续
- 类型开关：`switch x.(type)`

## 示例
```go
for i := 0; i < 3; i++ {
    fmt.Println(i)
}

switch day := 3; day {
case 1,2,3:
    fmt.Println("mid")
default:
    fmt.Println("other")
}
```
