# 07 接口与多态

## 特点
- 隐式实现：类型满足接口方法集即可
- 接口小而精：倾向组合多个小接口
- 零值：接口为 nil；注意 nil 接口与类型为 nil 的非空接口区分

## 示例
```go
type Notifier interface { Notify(msg string) error }

type Email struct{}
func (Email) Notify(msg string) error { fmt.Println("email", msg); return nil }

func Send(n Notifier, msg string) error { return n.Notify(msg) }
```

## 练习
- 定义 `Storage` 接口（Get/Set/Delete），实现内存版 map 存储
- 用接口描述日志输出（Console/File），演示多态
