# 02 变量、常量与基础类型

## 关键点
- `var` 显式声明 vs `:=` 推断
- 零值：数值 0，布尔 false，字符串 ""
- 常量与 `iota`
- 指针：`&v` 取址，`*p` 解引用，禁止指针算术

## 示例
```go
var a int = 10
b := 3.14
const Pi = 3.14159
const (
    A = iota // 0
    B        // 1
)

name := "Go"
p := &name
fmt.Println(*p)
```

## 练习
- 定义枚举型常量表示日志级别（Info/Warn/Error），使用 `iota`。
- 写一个函数返回多个值 `(sum, avg)`，演示短变量声明。
