# 01 快速开始与基本语法

## 目标
- 运行第一个 Go 程序
- 理解 `package main`、函数入口、导出规则

## 步骤
1. 创建文件 `hello.go`：
```go
package main
import "fmt"
func main() {
    fmt.Println("Hello, Go!")
}
```
2. 运行
```bash
go run hello.go
```
3. 构建
```bash
go build -o hello
./hello
```
