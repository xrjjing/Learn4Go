# 10 测试与基准

## 单元测试
- 文件后缀 `_test.go`，函数签名 `func TestXxx(t *testing.T)`
- 断言：`if got != want { t.Fatalf(...) }`

## 表驱动测试
```go
cases := []struct{ in, want int }{{1,2},{2,3}}
for _, c := range cases {
    if got := Inc(c.in); got != c.want {
        t.Errorf("%d -> %d, want %d", c.in, got, c.want)
    }
}
```

## 基准测试
- 函数签名 `func BenchmarkXxx(b *testing.B)`
- 运行：`go test -bench=. -benchmem`

## 覆盖率
- `go test -cover ./...`
