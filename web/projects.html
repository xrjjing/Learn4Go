项目实战

通过真实项目巩固所学知识，体验 Go 在不同场景下的应用。

CLI 批量重命名工具

一个实用的命令行工具，用于批量重命名文件，支持正则表达式匹配和 dry-run 模式。

flag os filepath
go run ./cmd/batchrename --help
日志归档工具

一个后台服务，定期扫描并压缩旧的日志文件，节省磁盘空间。

archive/zip goroutine time
go run ./cmd/logzip
TODO REST API

一个功能完整的 RESTful API 服务，用于管理待办事项列表，支持内存/SQLite/MySQL 三种存储。

net/http GORM MySQL
go run ./cmd/todoapi
Worker Pool 并发处理

展示 Go 并发能力的经典模式，使用 Goroutine 池高效处理大量并发任务。

goroutine channel sync.WaitGroup
go run ./examples/workerpool
gRPC 示例

一个简单的客户端/服务端示例，演示如何使用 gRPC 和 Protocol Buffers 定义和实现 RPC 服务。

gRPC Protobuf Streaming
go run ./examples/grpc/server
API 网关

使用 Gin 框架实现一个反向代理网关，作为微服务架构的统一入口，支持路由转发和日志记录。

Gin httputil.ReverseProxy Middleware
go run ./examples/gateway/gin
