快速开始

只需两步，即可在本地启动完整的学习环境。

# 1. 克隆项目
git clone https://github.com/your-repo/learn4go.git
cd learn4go

# 2. 本地启动（无需 Docker）
./start-local.sh

# 或使用 Docker 部署
cd deployments && docker-compose up -d

学习门户: http://localhost:8000/portal.html

学习路线

阶段一：Go 语言基础

掌握 Go 的基本语法、并发模型和工程管理。这是从 Java 到 Go 思维转变的关键一步。

阶段二：开发环境与框架

熟悉 Go 的工具链，学习 Gin、gRPC 等主流框架，为开发真实应用打下基础。

阶段三：项目实战

通过一系列精心设计的实战项目，将所学知识融会贯通，获得宝贵的开发经验。

阶段四：部署与运维

学习使用 Docker 进行项目打包、部署和管理，了解 Go 应用的生产环境实践。
