# 工具链与依赖管理

## 基础命令
- `go fmt ./...`：格式化
- `go vet ./...`：静态检查
- `go test ./...`：单元测试
- `go test -race ./...`：数据竞争检测

## 模块
- `go mod init <module>` 创建模块
- `go mod tidy` 整理依赖
- 代理：`GOPROXY=https://goproxy.cn,direct`

## 目录结构建议
```
.
├── cmd/           # 可执行入口（后续可添加）
├── internal/      # 内部包
├── pkg/           # 可复用包
├── 03.项目实战/   # 示例项目就地放置
```

## 练习
- 添加 `Makefile`：fmt、vet、test 三个目标
- 运行 `go test ./...` 验证示例
